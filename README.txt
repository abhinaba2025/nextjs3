1. Extract the downloaded zip file

2. Open terminal and navigate to the project folder:
   cd shopnex

3. Install dependencies:
   npm install

4. Start development server:

5. CUSTOMIZATION
================================================================================

COLORS
------
Edit tailwind.config.js to change the color palette:

Primary: #4F46E5 (Indigo)
Accent:  #F59E0B (Amber)
Dark:    #0F172A (Slate 900)
Light:   #F8FAFC (Slate 50)

FONTS
-----
Default font: Poppins
Update in index.html and tailwind.config.js

LOGO
----
Replace the logo SVG in src/components/layout/Header.tsx

PRODUCTS
--------
Edit product data in src/data/products.ts

PAGES
-----
Add or modify pages in src/pages/
Update routes in src/App.tsx

6. DEPLOYMENT
================================================================================

VERCEL (Recommended)
--------------------
1. Push code to GitHub
2. Go to vercel.com
3. Import repository
4. Click Deploy

NETLIFY
-------
1. Run: npm run build
2. Drag 'dist' folder to Netlify

STATIC HOSTING
--------------
Upload contents of 'dist' folder to any static host.

Note: Configure SPA routing (redirect all requests to index.html)
