  2. Add route in src/App.tsx

10. DARK/LIGHT MODE
================================================================================

ShopNex defaults to Light Mode. Users can toggle using the sun/moon icon
in the header. Theme preference is saved to localStorage.

Technical details:
  â€¢ ThemeContext manages global theme state
  â€¢ Toggle adds/removes 'dark' class on <html>
  â€¢ Tailwind dark: variant applies dark styles
  â€¢ CSS variables update colors per theme
  â€¢ Smooth 300ms transition between themes

To change default theme:
  Edit src/context/ThemeContext.tsx and change the initial state.
