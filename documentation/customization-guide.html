üé® Changing Logo

Replace the text logo with your own image or custom design.

Option 1: Use Image Logo

Add Your Logo File

Place your logo image in public/images/ folder. Recommended formats: SVG, PNG (transparent background). Recommended size: 150x40px for header.

Update Header Component

Replace the text logo with an image tag.

üìÑ src/components/layout/Header.tsx
// Replace the text logo with image <Link to="/" className="flex items-center"> <img src="/images/logo.svg" alt="YourBrandName" className="h-10 w-auto" /> </Link> // For dark/light mode support, use conditional rendering: <img src={isDark ? "/images/logo-light.svg" : "/images/logo-dark.svg"} alt="YourBrandName" className="h-10 w-auto" />

Option 2: Custom Text Logo with Different Font

// Use a custom font for text logo <Link to="/" className="flex items-center gap-2"> <span className="font-serif italic text-2xl font-bold"> YourBrandName </span> </Link>
‚ÑπÔ∏è
Logo Sizes
  • Header Logo: 150x40px (or height 40px, auto width)
  • Footer Logo: 180x50px
  • Mobile Logo: 120x32px

‚≠ê Changing Favicon

Update the browser tab icon (favicon) with your own design.

Prepare Your Favicon

Create favicon in multiple sizes: 16x16, 32x32, 180x180 (Apple Touch). Use PNG or SVG format.

Add Favicon Files

Place favicon files in public/ folder: favicon.ico, favicon-32x32.png, apple-touch-icon.png

Update index.html

Replace the existing favicon link tags.

üìÑ index.html
<!-- Replace existing favicon with your files --> <link rel="icon" type="image/x-icon" href="/favicon.ico"> <link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png"> <link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
üí°
Free Favicon Generator Use realfavicongenerator.net to generate all favicon sizes from a single image.

üìÑ src/index.css

Tailwind Color Classes

You can also use Tailwind color classes throughout components:

// Instead of: className="bg-indigo-600 hover:bg-indigo-700" // Change to: className="bg-blue-600 hover:bg-blue-700" // or className="bg-emerald-600 hover:bg-emerald-700"

üîç SEO Settings

Global SEO (index.html)

üìÑ index.html
<title>Your Brand - Your Tagline</title> <meta name="description" content="Your site description for search results"> <meta name="keywords" content="keyword1, keyword2, keyword3"> <!-- Open Graph for social sharing --> <meta property="og:title" content="Your Brand"> <meta property="og:description" content="Your description"> <meta property="og:image" content="https://yourdomain.com/og-image.jpg"> <meta property="og:url" content="https://yourdomain.com">
üí°
SEO Best Practices
  • Keep title under 60 characters
  • Description should be 150-160 characters
  • Use relevant keywords naturally
  • Create unique og:image (1200x630px)
