‚öôÔ∏è Installation

Follow these steps to install and run ShopNex locally:

Step 1: Extract Files

Extract the downloaded zip file to your desired location.

Step 2: Install Dependencies

Open your terminal, navigate to the project folder, and run:

cd shopnex
npm install

Step 3: Start Development Server

Run the development server:

npm run dev

The application will be available at http://localhost:5173

Step 4: Build for Production

To create a production build:

npm run build

The build output will be in the dist folder.

Success! Your ShopNex store is now running locally.

üé® Customization

Colors

Modify the color palette in tailwind.config.js:

// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#4F46E5',    // Indigo
        accent: '#F59E0B',     // Amber
        dark: '#0F172A',       // Slate 900
        light: '#F8FAFC',      // Slate 50
      }
    }
  }
}

Typography

The default font is Poppins. To change it, update index.html and tailwind.config.js.

Logo

Replace the logo in src/components/layout/Header.tsx.

Products

Edit product data in src/data/products.ts.

üåô Theming (Light/Dark Mode)

ShopNex includes a built-in theme toggle. The theme preference is saved to localStorage.

How it works

The ThemeContext manages the theme state. When toggled, it adds/removes the dark class on the HTML element.

// Using the theme hook
import { useTheme } from '@/context/ThemeContext';

function MyComponent() {
  const { theme, toggleTheme } = useTheme();
  
  return (
    <button onClick={toggleTheme}>
      {theme === 'light' ? 'Dark Mode' : 'Light Mode'}
    </button>
  );
}
üí°
The default theme is Light Mode. Users can toggle between themes using the sun/moon icon in the header.

üîå API Integration

ShopNex is designed to work with any headless CMS or eCommerce backend.

Recommended Backends

  • Shopify Storefront API - Full eCommerce solution
  • Saleor - Open-source GraphQL eCommerce
  • Medusa - Open-source headless commerce
  • Strapi - Headless CMS
  • Sanity - Real-time content platform

Example: Fetching Products

// src/services/api.ts
import axios from 'axios';

const api = axios.create({
  baseURL: process.env.VITE_API_URL,
});

export async function getProducts() {
  const { data } = await api.get('/products');
  return data;
}

export async function getProduct(slug: string) {
  const { data } = await api.get(`/products/${slug}`);
  return data;
}
