Colors & Branding

Update the CSS variables in src/index.css:

:root {
  --color-primary: #4F46E5;    /* Indigo - main brand color */
  --color-accent: #F59E0B;     /* Amber - accent/CTA color */
  --color-dark: #0F172A;       /* Dark background */
  --color-light: #F8FAFC;      /* Light background */
}

Logo

Replace the logo in src/components/layout/Header.tsx. Search for the ShopNex text and SVG icon.

Typography

The default font is Poppins. To change:

  1. Update the Google Fonts link in index.html
  2. Update the font-family in src/index.css

Products

Edit product data in src/data/products.ts. Each product has: id, name, slug, price, originalPrice, image, images, category, rating, reviews, badge, description, features.

Blog Posts

Edit blog data in src/data/blogData.ts. Each post has: id, slug, title, excerpt, content, category, author, date, readTime, image, tags.

Magazine Articles

Edit magazine data in src/data/magazineData.ts. Each article has: id, slug, title, subtitle, excerpt, content, category, author, date, readTime, image, tags.

Adding New Pages

  1. Create a new component in src/pages/
  2. Add the route in src/App.tsx
  3. Add navigation link in src/components/layout/Header.tsx

ğŸŒ™ Dark/Light Mode

ShopNex includes a fully functional dark/light theme toggle. The default is Light Mode. Theme preference is persisted in localStorage.

How It Works

  1. The ThemeContext manages theme state globally
  2. Toggling adds/removes the dark class on the HTML element
  3. Tailwind CSS dark variant applies dark mode styles
  4. CSS variables update colors for both themes

Using the Theme Hook

import { useTheme } from './context/ThemeContext';

function MyComponent() {
  const { theme, toggleTheme, setTheme } = useTheme();
  
  return (
    <button onClick={toggleTheme}>
      {theme === 'light' ? 'ğŸŒ™ Dark' : 'â˜€ï¸ Light'}
    </button>
  );
}

Customizing Dark Mode Colors

Edit the dark theme variables in src/index.css:

.dark {
  --color-bg: #0F172A;
  --color-surface: #1E293B;
  --color-text: #F1F5F9;
  --color-border: #334155;
}

â“ FAQ

How do I change the logo?

Edit src/components/layout/Header.tsx and replace the SVG icon and "ShopNex" text.

How do I add a new product?

Add a new product object to the products array in src/data/products.ts.

How do I add a new homepage demo?

Create a new component in src/pages/, add a route in App.tsx, and add it to the homeDemos array in Header.tsx.

Can I use this with a real payment gateway?

Yes! Integrate Stripe, PayPal, or any payment provider. The checkout form is ready for integration.

Does this work with Next.js?

This template is built with Vite + React. Migration to Next.js would require restructuring routes to the App Router format.

How do I deploy to a subdirectory?

Set the base option in vite.config.ts: base: '/subdirectory/'
